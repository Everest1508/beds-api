## URL

```http
https://bedsapi.pythonanywhere.com/
```
## Endpoints

#### Beds:

```http
GET /beds/: Retrieve a list of all beds
POST /beds/: Create a new bed
GET /beds/{id}/: Retrieve details of a specific bed
PUT /beds/{id}/: Update details of a specific bed
DELETE /beds/{id}/: Delete a specific bed
```

#### Patients:

```http
GET /patients/: Retrieve a list of all patients
POST /patients/: Create a new patient
GET /patients/{id}/: Retrieve details of a specific patient
PUT /patients/{id}/: Update details of a specific patient
DELETE /patients/{id}/: Delete a specific patient
```
